# 라이브러리 파일 다운로드 안내

이 폴더에 다음 두 파일을 다운로드하여 추가해주세요:

## 1. jsPDF 다운로드

**URL:** https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js

**다운로드 방법:**
1. 위 URL을 브라우저에서 열기
2. Ctrl+S (Windows) 또는 Cmd+S (Mac)로 저장
3. 파일명을 `jspdf.umd.min.js`로 저장
4. 이 `libs/` 폴더에 저장

## 2. html2canvas 다운로드

**URL:** https://cdnjs.cloudflare.com/ajax/libs/html2canvas/1.4.1/html2canvas.min.js

**다운로드 방법:**
1. 위 URL을 브라우저에서 열기
2. Ctrl+S (Windows) 또는 Cmd+S (Mac)로 저장
3. 파일명을 `html2canvas.min.js`로 저장
4. 이 `libs/` 폴더에 저장

## 최종 폴더 구조

```
libs/
 ├─ jspdf.umd.min.js      ← 다운로드 필요
 ├─ html2canvas.min.js    ← 다운로드 필요
 └─ README.md             ← 이 파일
```

## 확인 방법

두 파일을 다운로드한 후, 브라우저 콘솔에서 경고 메시지가 사라지면 정상입니다.
