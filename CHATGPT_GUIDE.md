# ChatGPT에게 요청할 내용

## ChatGPT에게 복사해서 붙여넣기:

```
나는 로컬에 있는 Git 저장소를 GitHub에 푸시하고 배포하려고 해요.

현재 상황:
- 로컬 폴더: /Volumes/김ssd/앱개발/diposture
- GitHub 저장소: https://github.com/skyman200/posture-ai-kor
- 원격 저장소는 이미 연결되어 있음 (git remote -v로 확인 가능)
- 모든 코드는 로컬에 커밋되어 있음
- GitHub 사용자명: skyman200

해야 할 일:
1. git push -u origin main 실행 (코드를 GitHub에 업로드)
2. npm run deploy 실행 (GitHub Pages로 배포)

문제: GitHub 인증이 필요함. 무료 방법으로 진행하고 싶어요.

GitHub CLI를 사용해서 인증하고 푸시 및 배포까지 완료해주세요.
만약 GitHub CLI가 설치되어 있지 않다면 설치하고,
gh auth login으로 브라우저 인증을 진행한 다음,
git push와 npm run deploy를 실행해주세요.
```

## 또는 더 간단하게:

```

GitHub Pages로 배포해줘.

GitHub CLI로 인증하고 진행해줘.
```
